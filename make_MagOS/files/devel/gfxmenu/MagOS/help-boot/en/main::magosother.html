Other parameters

There are two types of boot options available. First, there are options that affect the system. Second, there are kernel options. Some of the more common options are:

a) system options

  • Copy to RAM - System will be copied to RAM and will be started from memory. This require more than 2GB RAM.

b) kernel options

  • >Safe Setting - Hardware failsafe parameters. Select this if you encounter hangs while booting or irreproducible errors. This option disables DMA for IDE drives and all power management features.
